# 유가-코스피 상관관계 검증 결과 (Verifier 분석)

## 핵심 결론: 유가는 "최고 상관계수" 지표가 아님

### 실제 상관관계 순위 (KOSPI200 기준)
1. KOSPI200 야간선물 → 가장 직접적
2. S&P 500 / NASDAQ 선물 → 글로벌 리스크
3. USD/KRW 환율 → 외인 매매 직결 (-0.5 ~ -0.7)
4. VIX → 변동성/리스크
5. WTI 유가 → 4~5위 (+0.3 ~ +0.5, 레짐에 따라 반전)

### 유가 상관관계가 반전되는 경우
- 수요 주도 상승 (경기 좋음) → 유가↑ 코스피↑ (양의 상관)
- 공급 충격 (이란/전쟁) → 유가↑ 코스피↓ (음의 상관!)
- OPEC 결정 → 상관관계 약화

### USD/KRW가 유가보다 강한 이유
유가→코스피 경로: 유가 → USD/KRW → 외인매매 → 코스피
즉, 유가 효과의 상당 부분이 환율을 통해 매개됨
환율을 직접 모니터링하는 것이 더 효과적

### 백테스트 최소 기간
- 1~2개월: 통계적 유의성 없음 (❌)
- 6개월: 최소 잠정 결론 가능
- 2~3년: 전략 개발 권장
- 5년+: 복수 레짐 포함 필요
